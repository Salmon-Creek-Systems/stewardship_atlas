# Interactive Web Map Help

## Basic Navigation
- **click/drag** to move map
- **scroll** to zoom
- **alt+click** (desktop) or **long press** (mobile) to copy location

## Layer Management
- **control layer visibility** in legend box
- **select basemap** with dropdown

## Location Sharing
- Use the location sharing dropdown to choose format:
  - JSON coordinates
  - Google Maps link  
  - Internal map link
- Paste coordinates in the input field and click "Go" to navigate
